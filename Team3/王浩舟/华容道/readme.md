完成了基于iPhone 11屏幕界面设计的简单华容道游戏，实现的功能有：
1. 图形化华容道方块
2. 方块移动合法性判断
3. 游戏结束判断
4. 基于当前游戏地图一键完成
5. Spring动画实现

其中具体涉及到的技术细节包括：
1. 对话框的调用
2. UIView的使用
3. 划动手势识别
4. 动画的调用
5. 基本语言特性和算法（Struct、Hash、BFS）
